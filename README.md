# qrvc
qrvc is command line tool to prepare a QR code from a vCard.

## Install

When you have go installed on your machine, you can issue the followint command to install qrvc on your machine:

```sh
go install github.com/ulfschneider/qrvc@latest
```


## Develop

To work with the source code and build with the help of the Makefile, you first have to install the following dependencies:

```sh
go install golang.org/x/vuln/cmd/govulncheck@latest
go install github.com/CycloneDX/cyclonedx-gomod/cmd/cyclonedx-gomod@latest
go install github.com/google/go-licenses/v2@latest
```
